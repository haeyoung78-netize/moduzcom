# MODUZ (모두즈 아틀리에) - Web Application

본 프로젝트는 생성형 AI의 정교함과 인간 중심의 교육학적 깊이를 융합한 프리미엄 평생교육 설계 아틀리에 **MODUZ(모두즈)**의 고성능, 고대비 반응형 웹 애플리케이션입니다. 

Windows 환경의 VS Code에서 로컬 서버를 구동하고 실행하는 데 필요한 가이드와 프로젝트 구조를 아래와 같이 안내합니다.

---

## 📋 사전 준비 사항 (Prerequisites)

- **Node.js**: `v18.0.0` 이상 또는 `v20.x` (LTS 버전 권장)
- **npm**: `v9.x` 이상 (Node.js 설치 시 함께 설치됨)

---

## 🚀 로컬 개발 및 빌드 실행 가이드

### 1. 의존성 패키지 설치
프로젝트 루트 폴더에서 터미널을 열고 다음 명령어를 실행하여 필요한 패키지를 설치합니다.
```bash
npm install
```

### 2. 로컬 개발 서버 실행
설치가 완료된 후 아래 명령어로 개발용 로컬 서버를 실행합니다.
```bash
npm run dev
```
- 기본 설정에 따라 서버는 **`http://localhost:3000`**에서 실행됩니다.
- 웹 브라우저를 열고 해당 주소로 접속하면 고해상도의 디자인 화면을 실시간으로 확인하실 수 있습니다.

### 3. 프로덕션 빌드 (정적 파일 생성)
배포 가능한 최적화된 정적 파일을 빌드하려면 아래 명령어를 사용합니다.
```bash
npm run build
```
- 빌드가 완료되면 프로젝트 루트에 `/dist` 폴더가 생성되며, 고성능 경량 정적 에셋(HTML, JS, CSS)이 저장됩니다.

## 📂 주요 폴더 구조 (Project Directory)

```text
/
├── assets/                 # 정적 리소스 및 미디어 파일 폴더
├── src/                    # React 애플리케이션 소스 코드
│   ├── components/         # 독립된 개별 화면 컴포넌트들
│   │   ├── Home.tsx        # 메인 홈 화면 (브랜드 핵심 가치 및 하이라이트)
│   │   ├── About.tsx       # 아틀리에 소개 (철학, 미션, 페다고지)
│   │   ├── Courses.tsx     # 아카데미 교육 과정 (필터링 목록)
│   │   ├── CourseDetail.tsx# 개별 과정 상세 (커리큘럼 및 커리어맵)
│   │   ├── Insights.tsx    # 인사이트 블로그 (사이드바, 태그 검색형 대시보드)
│   │   ├── Reviews.tsx     # 실시간 수강 후기 (등록 및 필터링 기능)
│   │   ├── Store.tsx       # 디지털 스토어 (템플릿, 라이선스 전용 필터링 대시보드)
│   │   └── Contact.tsx     # 상담 및 도입 신청 양식 (인하우스/공공 위탁 신청)
│   ├── App.tsx             # 전역 레이아웃 및 페이지 네비게이션 라우터
│   ├── main.tsx            # React 진입점(Entry Point)
│   └── index.css           # 글로벌 CSS 및 Tailwind 테마 정의 (한글 폰트 적용)
├── index.html              # HTML 기본 템플릿 (웹폰트 및 아이콘 연동)
├── package.json            # 프로젝트 의존성 및 스크립트 정의
├── tsconfig.json           # TypeScript 빌드 설정
└── vite.config.ts          # Vite 환경 설정 파일 (정적 최적화 및 로컬 서빙)
```

## ✨ 구현된 화면 및 핵심 기능 (Implemented Features)

1. **Atelier Public Layout (일반 네비게이션 뷰)**
   - **홈(Home)**: 명확한 타이포그래피와 모션 효과를 갖춘 오프닝 시퀀스, 핵심 철학, 추천 강의 등 정밀 배치된 섹션.
   - **소개(About)**: 디자인과 교육의 융합을 상징하는 비주얼 카드 및 아틀리에 아키텍처.
   - **아카데미(Courses)**: 카테고리별(Corporate, Executive, Public) 과정 탐색 및 직관적 상세 이동 연동.
   - **과정 상세(CourseDetail)**: 실무형 주차별 커리큘럼, 맞춤 로드맵 정보 등 심도 있는 안내 레이아웃.
   - **수강 후기(Reviews)**: 실제 학습자의 리뷰 목록 조회, 과정별 실시간 필터링 및 **수강생 후기 직접 작성 폼** 구현 (제출 시 상단 목록에 즉시 추가).
   - **상담 신청(Contact)**: B2B/B2G 도입 및 상담 신청 폼 작성 및 개인정보동의 유효성 검사.

2. **Atelier Dashboard Layout (대시보드 전용 사이드바 뷰)**
   - **인사이트 블로그(Insights)**: 검색창, 태그 기반 고속 필터링 사이드바, 카테고리별 고급 아티클 아카이브 레이아웃.
   - **디지털 스토어(Store)**: 학습 도구, 교안 라이선스, 노션 템플릿 목록, 정렬 및 가격 슬라이더 필터링 대시보드.

## 🛠️ 아직 구현되지 않은 기능 (미구현 / 가상 처리 사항)

본 프로젝트는 고화질의 정교한 화면을 구현한 React 싱글 페이지 애플리케이션(SPA)으로, 실시간 프론트엔드 데이터 흐름을 충실히 검증할 수 있도록 설계되었습니다. 다만 아래의 실제 서버 및 데이터베이스 연동이 필요한 부분은 **화면 기획에 부합하는 안전한 가상 클라이언트 로직**으로 동작합니다.

- **실시간 데이터베이스 영구 저장**: 수강 후기 등록 및 상담 신청 시, 로컬 React 상태(State) 메모리에 즉시 업데이트되어 화면에서 즉각적인 결과 확인이 가능하나, 페이지 새로고침 시 초기화됩니다. (실무 배포 시 Firestore 등의 Cloud DB 연동이 권장됩니다.)
- **실제 결제 및 라이선스 파일 즉시 다운로드**: 스토어 내 상품의 "구매 및 라이선스 발급" 선택 시, 결제 완료 알림 모달이 트리거되어 UI의 흐름을 확인하는 가상 인터랙션으로 설계되어 있습니다.
- **실제 사용자 인증/회원가입**: 사이드바 하단 로그아웃 버튼 클릭 시, 게스트 모드 전환 알림과 함께 즉시 메인 화면으로 리다이렉트되는 가상 인증 시뮬레이션으로 동작합니다.
